== Description ==

This wordpress plug-in will put a link to each posts and pages to report that particluar post or page. The reported posts will be available in the backend.

== Installation ==

1. Upload the 'wp-report-posts.php' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings/options menu to configure this plugin
